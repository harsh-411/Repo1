Simpson's 1/3 Rule Integration

Calculate the integral of a function using Simpson's 1/3 rule.
Note: The number of subdivisions (n) must be even.
You can use trigonometric functions (e.g., sin(x)) or exponential functions (e.g., exp(x)).
