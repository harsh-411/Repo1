Euler's Method ODE Solver

Solve the initial value problem dy/dx = f(x, y) with y(x₀) = y₀ using Euler's method.
Provide the initial x (x₀), initial y (y₀), final x (Xₙ), and step size (h).
(You can use shorthand for functions like sin(x) or exp(x); these are automatically converted.)

Project Description & Formula

Euler's Method:
To solve the ODE dy/dx = f(x, y) with y(x₀) = y₀, Euler's method uses:

yₙ₊₁ = yₙ + h · f(xₙ, yₙ)

How It Works:
1. Enter the initial values x₀ and y₀, the final x (Xₙ), and the step size (h).
2. The differential equation f(x, y) is dynamically generated from your input (shorthand for functions like sin(x) or exp(x) are automatically converted).
3. At each step, Euler's method computes the new y value and records iteration details.
4. The final approximate value of y at Xₙ is displayed, along with a table of all iteration data that can be toggled.

This project demonstrates an ODE solver using Euler's method with dynamic theming, customizable accent colors, detailed iteration tracking, and support for trigonometric and exponential functions.
