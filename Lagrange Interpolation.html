Lagrange Interpolation

Interpolate a value at Xₙ using the Lagrange formula.
Enter x-values and y-values as comma‑separated lists (ensure both lists have the same number of values), and then provide the target X value.

Project Description & Formula

Lagrange Interpolation Formula:
The interpolated value at Xₙ is given by:

L(Xₙ) = Σ [ yᵢ · Lᵢ(Xₙ) ], where
Lᵢ(Xₙ) = ∏j≠i ((Xₙ − xⱼ)/(xᵢ − xⱼ))

How It Works:
1. Enter x-values and y-values as comma‑separated lists and provide the target X value.
2. For each data point, the program computes:
- The numerator: the product of (Xₙ − xⱼ) for j ≠ i
- The denominator: the product of (xᵢ − xⱼ) for j ≠ i
- The Lagrange basis value Lᵢ(Xₙ) = (numerator)/(denominator)
- The term contribution: yᵢ · Lᵢ(Xₙ)
3. The sum of these terms gives the interpolated value at Xₙ.
