Newton-Raphson Root Finding

Find an approximate root of f(x) using the Newton-Raphson method.
Provide an initial guess, tolerance, and maximum iterations.
(You can use functions like sin(x) or exp(x); shorthand will be automatically converted.)

Project Description & Formula

Newton-Raphson Method:
The method computes successive approximations to the root of f(x) using:
xₙ₊₁ = xₙ - f(xₙ)/f'(xₙ)

How It Works:
1. Enter an initial guess, tolerance, and maximum iterations.
2. The derivative f'(x) is approximated numerically using a small delta (e.g., 1e-6).
3. Iteration details (iteration number, current x, f(x), derivative, and new x) are recorded.
4. The approximate root and number of iterations are displayed.

This project demonstrates the Newton-Raphson method for root finding with dynamic theming, customizable accent colors, detailed iteration tracking, and support for trigonometric and exponential functions.
