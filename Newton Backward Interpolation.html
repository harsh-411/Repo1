Newton Backward Interpolation

Enter x‑values and y‑values as comma‑separated lists, and provide the target X value.
(The target X should be near the last data point.)

Project Description & Formula

Newton Backward Interpolation:
Given equally spaced data points x₀, x₁, …, xₙ (with h = x₁ − x₀) and corresponding y‑values, compute u = (X – xₙ)/h and interpolate using:

P(X) = yₙ + u·Δyₙ + [u(u+1)/2!]·Δ²yₙ + [u(u+1)(u+2)/3!]·Δ³yₙ + …

How It Works:
1. The program computes the full forward difference table from the input data.
2. It extracts the backward differences (the last element from each row) and computes the interpolation polynomial using the Newton Backward formula.
3. Detailed iteration information is shown below, including for each order: the backward difference, term contribution, cumulative sum, and the formula used.
4. Additionally, the full forward difference table is displayed.

This project demonstrates Newton Backward Interpolation with dynamic theming, customizable accent colors, and detailed iteration tracking.
