Project Description & Formula

Newton Divided Difference Interpolation:
Given a set of data points (x₀, y₀), (x₁, y₁), …, (xₙ, yₙ), the divided difference table is constructed as follows:

f[x₀] = y₀
f[x₀,x₁] = (f(x₁) - f(x₀))/(x₁ - x₀)
f[x₀,x₁,x₂] = (f[x₁,x₂] - f[x₀,x₁])/(x₂ - x₀)
… and so on.

The interpolation polynomial is then given by: P(x) = f[x₀] + (x - x₀)f[x₀,x₁] + (x - x₀)(x - x₁)f[x₀,x₁,x₂] + …
