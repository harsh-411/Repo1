Adams Predictor-Corrector ODE Solver

Solve the initial value problem dy/dx = f(x, y) with y(x₀) = y₀ using the Adams Predictor-Corrector method.
For x beyond the first four points (bootstrapped via RK4), the predictor is:
yₙ₊₁^(p) = yₙ + (h/24)[55fₙ − 59fₙ₋₁ + 37fₙ₋₂ − 9fₙ₋₃]
and the corrector is:
yₙ₊₁ = yₙ + (h/24)[9f(xₙ₊₁, yₙ₊₁^(p)) + 19fₙ − 5fₙ₋₁ + fₙ₋₂].
Provide the initial x (x₀), initial y (y₀), final x (Xₙ), and step size (h).
(You can use shorthand for functions like sin(x) or exp(x); these are automatically converted.)

Project Description & Formula

Adams Predictor-Corrector Method:
To solve the ODE dy/dx = f(x, y) with y(x₀) = y₀, first bootstrap four points using RK4. Then for n ≥ 3:

Predictor: yₙ₊₁^(p) = yₙ + (h/24)[55f(xₙ, yₙ) − 59f(xₙ₋₁, yₙ₋₁) + 37f(xₙ₋₂, yₙ₋₂) − 9f(xₙ₋₃, yₙ₋₃)]
Corrector: yₙ₊₁ = yₙ + (h/24)[9f(xₙ₊₁, yₙ₊₁^(p)) + 19f(xₙ, yₙ) − 5f(xₙ₋₁, yₙ₋₁) + f(xₙ₋₂, yₙ₋₂)]

How It Works:
1. Enter initial values (x₀, y₀), final x (Xₙ), and step size (h).
2. The first four values are computed using RK4.
3. For subsequent steps, the predictor-corrector formulas are applied to compute yₙ₊₁.
4. Iteration details (including the predicted and corrected y values) are recorded and can be toggled.
5. The final approximate value of y at Xₙ is displayed.

This project demonstrates an ODE solver using the Adams Predictor-Corrector method with dynamic theming, customizable accent colors, detailed iteration tracking, and support for trigonometric and exponential functions.
