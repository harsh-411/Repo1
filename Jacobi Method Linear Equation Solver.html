Jacobi Method Linear Equation Solver

Solve a 3×3 system of linear equations using the Jacobi iteration method.
Enter the coefficients of the 3×3 matrix (aij) and the right‑hand‑side values (b₁, b₂, b₃).
Also provide an initial guess for x, y, and z, a tolerance, and a maximum number of iterations.

Project Description & Formula

Jacobi Method:
To solve the linear system A·x = b, the Jacobi iteration updates each variable using:

xi^(new) = (bi − Σ (aij · xj^(old)) for j ≠ i) / aii

How It Works:
1. Enter the coefficients for the 3×3 matrix and the right‑hand‑side values.
2. Provide an initial guess for the solution vector (x, y, z), a tolerance, and a maximum iteration count.
3. The method iteratively computes new approximations for x, y, and z until the solution converges within the tolerance.
4. Detailed iteration information is available by toggling the iteration details table.
