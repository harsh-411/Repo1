Newton Forward Interpolation

Perform Newton Forward Interpolation using equally spaced data points.
Enter x‑values and y‑values as comma‑separated lists (ensure equal lengths), and provide the target X value.

Project Description & Formula

Newton Forward Interpolation:
Given equally spaced data points with x₀, x₁, …, xₙ (with h = x₁ − x₀) and corresponding y‑values, compute u = (X – x₀)/h and interpolate using:

P(X) = y₀ + u·Δy₀ + [u(u–1)/2!]·Δ²y₀ + [u(u–1)(u–2)/3!]·Δ³y₀ + …

How It Works:
1. The program computes the full forward difference table (all orders).
2. It then evaluates the interpolation polynomial using the first element of each row.
3. The detailed iteration table below now includes for each order the forward difference values, the term contribution for that order, the cumulative sum (the running value of the polynomial), and the formula used for that term.
