Milne Predictor and Corrector

Enter initial x‑values, y‑values, and a custom function for f(x, y).
(Example: x: 0,0.2,0.4,0.6; y: 0,0.02,0.0795,0.1762; f(x,y): x - y*y)
You can use trigonometric functions like sin(x), cos(x) and others (they will be replaced with Math.sin(x), etc.).

Project Description & Formula

Milne Predictor and Corrector:
For an ODE defined by f(x, y) = custom function, Milne’s method uses current data points to predict and correct the next y value.

Milne Predictor:
MP = f(x₀, y₀) + (4h/3) · (2f(x₁, y₁) – f(x₂, y₂) + 2f(x₃, y₃))

Milne Corrector:
MC = y₂ + (h/3) · (f(x₂, y₂) + 4f(x₃, y₃) + f(x₄, MP))

The computed next value is appended to the arrays, and detailed iteration information is displayed.

Dynamic theming and accent color options are provided for a customizable interface.
