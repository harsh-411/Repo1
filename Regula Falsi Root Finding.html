Regula Falsi Root Finding

Find an approximate root of f(x) using the Regula Falsi (False Position) method.
Enter two initial guesses (a and b) such that f(a)*f(b) < 0, along with a tolerance and maximum iterations.
(You can use functions like sin(x) or exp(x); shorthand will be automatically converted.)

Project Description & Formula

Regula Falsi Method Formula:
Given two initial points a and b such that f(a)*f(b) < 0, the next approximation is computed as:
c = (a·f(b) - b·f(a)) / (f(b) - f(a))

How It Works:
1. Enter two initial guesses (a and b) that bracket a root (f(a)*f(b) < 0), along with a tolerance and maximum iterations.
2. At each iteration, compute c using the formula above and update a or b depending on the sign of f(c).
3. The method stops when |f(c)| is less than the tolerance.
4. All iteration details (iteration number, a, b, c, f(a), f(b), and f(c)) are recorded and can be viewed.
5. Navigation buttons allow you to toggle between the calculator and description views.
